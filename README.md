# ordercentral_owner_app_flutter

Run Commands
Run in Development Environment
flutter run --flavor development --dart-define=env=development

Run in UAT Environment
flutter run --flavor uat --dart-define=env=uat

Run in Production Environment
flutter run --flavor production --dart-define=env=production

Build APK Commands
Build for Development Environment
flutter build apk --release --flavor development --dart-define=env=development

Build for UAT Environment
flutter build apk --release --flavor uat --dart-define=env=uat

Build for Production Environment
flutter build apk --release --flavor production --dart-define=env=production
